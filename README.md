# 검단 파라곤 — 계약자 라운지

검단 파라곤 아파트 계약자 커뮤니티의 데일리 브리핑 앱입니다.

## 기능

- 📅 날짜별 데일리 브리핑 조회
- 💬 토픽별 댓글 수 표시
- 🎯 카테고리별 색상 분류
- 🔥 HOT 토픽 표시
- 📝 토픽 상세 조회 및 라운지 댓글 확인

## 설치 및 실행

```bash
npm install
npm run dev
```

개발 서버는 `http://localhost:3000`에서 실행됩니다.

## 빌드

```bash
npm run build
```

## 구조

- `src/App.jsx` - 메인 컴포넌트
- `src/data.js` - 데이터
- `src/App.css` - 스타일
- `public/` - 정적 파일

## 기술 스택

- React 19
- Vite
- CSS-in-JS
